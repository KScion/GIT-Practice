HTML Tags:

HTML tag is a set of characters that constitutes commands in a web page and provides the direction and specifications for the web browser that how it must format and display the content. There is a beginning tag comprises of names and attributes and starts a specific command; whereas the ending tag comprises the same name but additionally "/" symbol at the front and closes/stops that particular command for any attribute.
